# 提示词画廊 · 对你的 Agent 直接说这些

> 学自 OpenMontage 的 DX 思想:好插件不该让人研究怎么用,复制一句话就能开工。

## 零配置就能跑(不需要任何 API key)
- 「素材在 ~/Desktop/素材,剪个 15 秒发抖音,唯美一点」
- 「把这 5 条视频按 拆箱→细节→上身 的节奏拼成 20 秒,配字幕」
- 「给这条 cutlist 出片:workspaces/xx/cutlist.json,加硬字幕和闪白」

## 配了 ARK_API_KEY 之后(生成能力)
- 「我想拍一条工艺科普,先出分镜给我看」
- 「按这个分镜生成 AI 参考片,480p 就行,生成前把预算报我」
- 「这段实拍质量不行,用 AI 生成一段空镜顶上(1080p,记得审帧)」

## 进阶
- 「用重音对齐把切点吸到这条 BGM 的真实鼓点上」
- 「previz 我看过了,S3 换成绸缎飘动,其它保留,重拼给我」
- 「起引擎服务,把剪单物化成剪映草稿,转场滤镜都要」
- 「make doctor 看看环境缺什么」

## 红线内玩法(这些它会拒绝/停下)
- 自动发布 ❌(永远停在"待人确认")
- 没给人看分镜就烧钱生成 ❌
- 剪单里写没有理由的刀 ❌(每刀必须 rationale)
