# 分镜方法论(video-storyboard 的 references)

> 通用方法层:不含任何品牌信息。品牌资料由调用方注入,技能只消费。

## 1. 三秒钩子类型库(段1专用)

| 钩子类型 | 打法 | 瞄准的互动(对齐热度分权重) |
|---|---|---|
| 认知冲突 | 「你以为的X其实是Y」,画面先给反常识细节 | 评论(×4):观众来纠正/站队 |
| 结果前置 | 先给成品/上身最惊艳一帧,再倒叙 | 收藏(×1)+完播 |
| 痛点直击 | 说出目标人群正在忍的问题 | 评论(×4):+1式共鸣 |
| 悬念留白 | 「最后一步 90% 的人不知道」 | 完播+关注(×8):追后续 |
| 身份点名 | 「(某类人群)别划走」 | 精准完播;慎用,易俗 |
| 视觉奇观 | 工艺/材质的罕见画面直接怼脸 | 收藏+转发(×4) |
| 系列悬念 | 「这是XX系列第N件,上一件…」 | 关注(×8):追账号 |

规则:钩子段 ≤3s;一条只用一种钩子;钩子许诺的,正片必须兑现(不做标题党——护栏是爆款率不是热度分单项)。

## 2. 结构模板(按内容类型)

- **ootd/上身展示**(4-5段,20-35s):钩子(结果前置/视觉奇观)→全身走动→细节特写×2(面料/剪裁)→氛围收口+关注钩
- **工艺科普**(4-6段,25-45s):钩子(认知冲突)→工艺特写→原理一句话→上身对照(工艺落在效果上)→收藏钩收口
- **对照解说**(文物/出处类,4-5段):钩子(原件图怼脸)→原件↔实物对照→细节迁移点→穿着场景→系列悬念收口
- **口播**(3-4段):结论前置→论据(配画面切换)→示例→行动指令(评论区提问)
- **开箱/上新**(4段):悬念钩→开箱动作→第一反应特写→上身即收口

## 3. 词表

- **景别**:远景/全景/中景/近景/特写(一段一景别,相邻段避免同景别连跳)
- **运镜**:固定/缓推/缓拉/横移/跟随/环绕/俯仰;手机实拍优先 固定+缓推+跟随(稳)
- **first_frame_hint 枚举**(对齐素材表12类):商品图/形象图/细节图/上身视频截帧/上身图/搭配图/空镜·场景/工艺特写/制作过程视频截帧/面料·纹理特写/文物原图/文物对照图

## 4. 配时法则

钩子≤3s;单段2–8s;总时长按平台:小红书 15–45s(默认25s)、视频号 30–60s;
信息密度:每段一个信息点,台词一段≤2句;留一段"无台词纯氛围"给BGM呼吸(通常倒数第二段)。

## 5. previz_prompt 写法

`主体(模特/服装的客观描述)+动作+运镜+光线+质感`,一段一镜头。
- 写「深色提花面料的长裙」不写品牌名/系列名(生成模型不需要,也防误用)
- 不写宣称词(高级/正宗/文物同款),质感用可见描述(光泽/垂坠/纹理)
- 时长≤5s/段(成本闸);实拍能拍出的画面才写(previz是拍摄标准,不是特效秀)

## 6. 宣称敏感词(校验器 warning 触发,输出进 claims_to_review 由人审)

绝对化:最/第一/顶级/100%/永不;功效:治/防病/瘦/抗菌(无检测报告不许);
真伪判定:正宗/正统/唯一传承(命门维,一律转人工——参见评论区禁答清单同源纪律)。

## 7. 防搬运

参考爆款只进方法论层(用哪种钩子/什么结构),台词字幕全原创;
若调用方给了拆解情报(explosive-teardown 输出),只允许使用其中的占位符化模板,同样不许出现原句。
