# video-storyboard · 选题 → 可拍摄分镜

> 一项**纯能力**:`主旨+资料+时长 → 分镜脚本(人读表 + 机器 JSON)`。
> 方法论见 `references/storyboard-method.md`(钩子类型库/结构模板/景别运镜词表/配时法则/previz_prompt 写法)。
> 它是视频链的**大脑上游**:video-previz 吃它的 JSON 生成示例片,video-editing 吃它的字幕进剪单。

## 红线(铁律)

1. **防搬运**:参考爆款只允许使用拆解情报的**抽象方法论**(钩子类型/结构/节奏),台词字幕**全部原创**,绝不复用任何参考视频/笔记的原句(沿用 explosive-teardown 的占位符纪律)。
2. **宣称合规**:台词/字幕不得出现绝对化用语、功效担保、医疗宣称;材质工艺的**事实性表述只能来自调用方提供的产品资料**,没有资料就写「[待补:xx]」占位,绝不编造。
3. **每段一镜头一意图**:单段只承载一个画面动作 + 一个信息点,段时长 2–8s,前 3 秒必须是钩子段。
4. **机器契约不省略**:每段 `previz_prompt / duration / first_frame_hint` 必填——下游程序靠它,不许写"同上"。
5. **旁白是显式决策不是默认留空**:分镜给人过目时必须写明旁白形态——「带口播(逐段分配)」或「无旁白·音乐驱动(理由)」,纯画面设计不许静默,用户可否决(2026-07-04 用户反馈)。

## 输入

- **必备**:选题或文案主旨;内容类型(ootd/工艺科普/对照解说/口播/其它——**模板是参考不是围栏**,类型不在列就按最近的模板改制并说明)
- **可选**:目标时长(缺省 25s)、产品资料(卖点/材质/工艺——由调用方从品牌库注入,技能不进库翻)、
  爆款拆解情报(方法论层)、平台(默认小红书)、账号人设、已有素材清单(**有什么拍什么**,分镜别要拍不出的画面)

## 工作流

### 1) 定骨架
按 `references/storyboard-method.md` 选该内容类型的结构模板,定段数(默认 3–6)与配时;
钩子段的意图要**指向互动权重**(想要评论→留争议点,想要关注→留系列悬念;对齐热度分 评论×4/关注×8)。
**完成判据**:骨架表(段号/意图/配时)成形,总时长 = 目标 ±20%。

### 2) 填分镜
每段填全:意图 / 景别 / 画面 / 运镜 / 台词(口播或旁白) / 字幕 / 音乐音效提示 / duration /
first_frame_hint(用素材表 12 类里的哪类图) / previz_prompt(镜头语言化:主体+动作+运镜+光线+质感,**不带品牌名与宣称**)。

### 3) 双格式落盘 + 校验闸
存进工作台 `素材库/{SKU}/04-工作台/{排期编号}/storyboard.md`(人读)与 `storyboard.json`(契约,见下;
工作区形态与 manifest 登记见 `$VE/README.md`),然后:
```bash
python3 skills/video-storyboard/scripts/validate_storyboard.py <json路径>
```
**完成判据**:校验器 `✓ 通过`(它查:段序/必填/段时长2-8s/总时长偏差/钩子段≤3s;宣称敏感词是 warning,提示人审)。

### 4) 交接
汇报分镜表 + 两个文件路径;告知下游:
`video-previz` 直接用 JSON 里每段的 `{seq, previz_prompt→prompt, duration, first_frame(按hint从素材库选)}`;
拍摄按段拍、按 `{SKU}-{排期编号}-S{n}_实拍.mp4` 回传;字幕列将进 video-editing 的剪单。
落库(贴进笔记「脚本·分镜」或独立 sheet)由调用方完成,本技能不碰飞书。

## 输出契约 JSON

```json
{"note_id": "PQ-012", "content_type": "工艺科普", "platform": "小红书", "total_duration": 25,
 "hook_type": "认知冲突", "claims_to_review": ["<涉宣称待人审的句子>"],
 "segments": [{"seq": 1, "intent": "钩子:制造认知冲突", "shot_size": "特写", "scene": "<画面>",
   "camera": "缓推", "line": "<台词>", "subtitle": "<字幕>", "sound": "<BGM/音效提示>",
   "duration": 3, "first_frame_hint": "工艺特写", "previz_prompt": "<镜头语言化生成提示词>"}]}
```
