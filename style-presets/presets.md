# 手艺选型的判断规则(不是答案清单)

> **唯一的硬约束**:剪单里的转场/滤镜/动画名必须存在于同目录 `enums.json`(机器 dump 的引擎全量枚举,
> upstream 升级后重新生成)。名字以外的一切选择都是 agent 的现场判断,按下面的规则做、照旧写 rationale。

## 怎么选滤镜
1. 先看素材:光线(顺/逆/平)、主色调、内容基调(唯美/知识/生活感),再去 enums.json 里找语义相近的名字试。
2. 全片只用一种滤镜;分段换滤镜=事故。
3. 强度从中档(60-75)起步:太高吃掉素材本色,产品类内容尤其要保真(面料/工艺的颜色是卖点,不许滤镜改色)。
4. 判断依据写进 rationale(例:"逆光暖调素材→选暖色滤镜压高光")。

## 怎么用转场
1. 转场只放**慢段之间**;卡点拍上的切换永远硬切——转场会吃掉拍点。
2. 一条 15-30s 的片:转场种类 ≤2 + 硬切;每加一种要有理由。
3. 强表现力的转场(水墨/快闪/分割类)一条片 ≤1 次,放在章节转折,不放开头结尾。
4. 转场时长 0.2-0.5s;超过 0.5s 就在偷时长,先想想是不是剪点本身有问题。

## 怎么用关键帧(缓推等)
1. 静态构图的段(空镜/定格/平铺)才需要缓推制造呼吸感,幅度 uniform_scale ≤1.08;有真实运镜的段不叠加。
2. 其它关键帧属性(位移/旋转/透明度)默认不用,分镜明确要求才上。

## 兜底纪律
- 拿不准名字是否存在:查 enums.json,没有就换,不猜。
- 拿不准效果:选保守项(硬切/低强度/不加),把犹豫写进 rationale 留人审。
- 本文件只写"怎么判断";若将来沉淀出"某内容类型验证有效的组合",应来自实拍数据复盘,记进品牌库,不写死在这里。
